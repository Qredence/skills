# Figma Agent Skills

This is Qredence’s active catalogue of Figma design and product skills.

Install skills with:

```bash
npx skills@latest add qredence/skills
```

Each directory is one skill. Its documents serve two surfaces:

- `SKILL.md` is the canonical document used by `skills.sh`.
- `SKILLS.md` is the identical document for direct upload through **Add skill** in Figma Design Agents.

Browse the directories to find skills for accessibility, design systems, components, prototyping, research, content, and handoff.
